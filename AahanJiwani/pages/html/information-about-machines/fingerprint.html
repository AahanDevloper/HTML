Information about Fingerprint scanner


A Fingerprint

It was discovered that Fingerprints of all human beings differ.

Fingerprint scanner was invented by Sir Francis Galton

The machine first converts the Fingerprint into a binary code which is then verified by that machine. Once the code is matched, we can say that it is verified that the device is unlocked by the right person. This is how fingerprint scanner plays a very important role in the feild of security, privacy and safety.
